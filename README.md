# CabBuddy – Campus Carpool System

CabBuddy is a console-based application that helps students organize carpool rides efficiently using core Data Structures.

## Features

• Student registration and authentication
• Ride request system (pickup, destination, time)
• Automatic ride matching using Priority Queue
• Carpool group formation
• Ride history tracking with Undo/Redo (Stack)
• Route mapping using Graph

## Data Structures Used

• Hash Map
• Queue
• Priority Queue
• Linked List
• Stack
• Graph

## Tech Stack

• C++
• Object-Oriented Programming
• File Handling

## How to Run

1. Compile:
   g++ code.cpp -o cabbuddy

2. Run:
   ./cabbuddy
